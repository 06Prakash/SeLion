Working with SeLion element events

How to hook into element events provided by SeLion.

Introduction

SeLion element events are those which are related to some element interaction, such as a "click", a "select", etc performed through an element object included with SeLion.

Getting Started with Element Event Handling in SeLion

SeLion offers support to hook into events that are triggered by SeLion elements. These steps can be followed to hook into a SeLion element event. As an example we are going to create a DebugScreenshotListener which takes a screenshot and adds it to the report before any element is clicked on.

  1. Create a new listener class by implementing the SeLion exposed interface ElementEventListener. Or extend the AbstractElementEventListener if you don't want to implement all methods.
