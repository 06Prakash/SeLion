How to create a project from SeLion archetype.

Prerequisites

Make sure to install the required software on your machine before you start using SeLion:

  1. Browser automation prerequisites
  2. Mobile automation prerequisites

Create Project from SeLion archetype

Once you have satisfied all of the prerequisites, open a command prompt and navigate to the directory where you want your project to be located, and run:


    

    
The sample project will automatically be generated from the SeLion archetype. You will see something like this:


    

    
Now run the following command in the project directory path from a command prompt.

    
mvn dependency:resolve

    
This command causes Maven to start downloading all of the required jar files.

    
Now run the following command in the project directory path from a command prompt.

    
mvn clean test

    
This command cleans the project temporary resources and runs the tests in the TestNG suite file SampleSuite.xml from the folder {project}/src/test/resources.
        This suite file is configured to run only the non user interface tests from the project.


    

    
Congratulations! You just created the project from SeLion archetype and executed the tests!
