Working with WebDriver events using SeLion

How to hook into events generated by WebDriver using SeLion.

Introduction

WebDriver events are primarily those events which are related to some action that Selenium WebDriver performs on the web page or on a native application such as "click" (or) "type" etc., Selenium's answer to hooking into such events is via EventFiringWebDriver. However, there are a couple of issues with using EventFiringWebDriver.

  1. It does not work with Actions
  2. The Selenium dev community is likely to deprecate this class out of the core libraries and perhaps relocate it to a different project -- possibly also in a different form.

Getting Started with Event Handling in SeLion

SeLion offers support to hook into events that are generated by WebDriver (including the ones that originate from the Actions class) in a different way. These steps can be followed to start hooking into WebDriver events using SeLion.
