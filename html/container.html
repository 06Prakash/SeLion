Rules for Using Containers

    
    
      
  • The name of the container element should end with the text Container. This
        is how SeLion identifies the type of an element when it resorts to generating actual java classes in Runtime.
    • 
      
  • Nesting of containers is NOT allowed. You cannot embed one container within the other.
