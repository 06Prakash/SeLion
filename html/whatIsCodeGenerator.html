Code Generator Maven Plugin

SeLion's Code Generator Maven Plugin will auto-generate Java PageObject class files from YAML during the Maven generate-sources phase.

Benefits of Using the SeLion Code Generator Maven Plugin

If you have web pages that change frequently, this is your secret sauce to scaling your test efforts!

The YAML page object file contains all the information it needs to map out a page. A change in your web page only requires a small change to YAML text. In addition, the YAML page object file allows you to define different identifiers based on locale.

Automating in different languages just got much easier!

If you're working across multiple teams, you can share the generated page classes as a packaged artifact. This allows other teams stay updated and does not impact test cases for every minor change.

Generated page classes can even extend other page classes for maximum reuse.

    2. 
    
  2. Once created, move the file into the directory
