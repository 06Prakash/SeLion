SeLionDataProvider

SeLionDataProvider is an extension to the concept of DataProvider introduced by TestNG

DataProvider in TestNG

TestNG provides us with the @DataProvider annotation to mark a method as supplying data for a test method.

It becomes extremely useful if we have one or more test cases, having the same logic, but needs to be run on different data. Any method annotated by a @DataSource may return an Object[][] array or Iterator<Object[]> that can be used by any Test annotated method to operate on the data. The DataProvider support in TestNG is generic and versatile enough to allow us to plug in any sort of data source or generator.

To learn more about the TesNG data providers please refer TestNG DataProviders.

SeLionDataProvider

SeLion provides SeLionDataProvider that extends the concept of DataProvider to help the users with the plumbing code. There are a variety of implementations of SeLionDataProvider that cater to the most commonly used data source file formats listed below.

  • YAML data source files.
  • Excel data source files.
  • XML data source files.
  • JSON data source files.

To get more information on a specific SeLionDataProvider refer to the links on the right panel.
